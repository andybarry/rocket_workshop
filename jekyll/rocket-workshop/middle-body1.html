Start the fuselage

in which we extrude sketches

{% include rocket-nav.html title="Fuselage Tutorial" num_pages="3" %}

1. Create a new part studio

  • Click the "+" button
  • Select Create Part Studio
{% include tutorial-video.html path="videos/middle-body/create_part_studio" %}
{% include section.html param="top" %}

2. Rename your part studio

  • Name it "Middle Body"
    • You're already thinking of skipping this... but then your future self will be sad...
{% include section.html %}

3. Create a sketch

  • Select the sketch tool
  • Select a sketch plane. Use the Top plane.
  • Align your view to the sketch plane by clicking "Top" on the view cube
  • Make sure you have selected the Top plane. In the upper right, you should see "Top Plane" under sketch plane
{% include tutorial-video.html path="videos/middle-body/create_sketch_top_plane" %} {% include section.html %}

4. Draw the outer body circle

  • Select the circle tool
  • Draw a circle. Be sure to snap it to the origin.
  • Select the dimension tool
  • Click on your circle and set its diameter to 0.781
{% include tutorial-video.html path="videos/middle-body/draw_outer_circle" %} {% include section.html %}

5. Draw the inner circle

  • Select the circle tool
  • Draw a circle inside your first circle. Be sure to snap it to the origin.
  • Select the dimension tool
  • Click first on the outer circle and then on the inner circle and set the difference between them to 0.025
{% include tutorial-video.html path="videos/middle-body/draw_inner_circle" %} {% include section.html %}

6. Extrude the body

  • Exit the sketch by clicking the green check
  • Select the extrude tool
  • Select "Sketch 1" from the left pane
  • Set the Depth to 3.0 in
  • Click the green check
{% include tutorial-video.html path="videos/middle-body/extrude_body" %}
