Assembly

in which we put it all together

{% include section.html param="top" %}

{% include step_counter.html %} Select the assembly button tab

  • Click the "Assembly 1" tab on the bottom row
{% include tutorial-video.html path="videos/assembly/select_assembly" %} {% include section.html %}

{% include step_counter.html %} Insert parts

  • Select the "Insert" tool
  • Select the "Middle body"
  • Click inside the assembly to place the part
  • Repeat for the "Lower body" and "Nosecone"
  • Click the green check
{% include tutorial-video.html path="videos/assembly/insert_parts" %} {% include section.html %}

{% include step_counter.html %} Mate the fuselage to the booster and fins

  • Select the "Fastened Mate" tool
  • Hover over the top of the booster and fins, then click when a disc appears
  • Hover over the inside of the fuselage, where the part gets wider
  • Click again, and you'll see the parts snap together
  • Click the green check
{% include tutorial-video.html path="videos/assembly/mate_booster_fuselage" %} {% include section.html %}

{% include step_counter.html %} Mate the fuselage to the nosecone

  • Select the "Fastened Mate" tool
  • Hover over the top of the fuselage then click when a disc appears
  • Hover over the inside of the nosecone, where the part gets wider
  • Click again, and you'll see the parts snap together
  • Click the green check
{% include tutorial-video.html path="videos/assembly/mate_nosecone" %} {% include section.html %}

{% include step_counter.html %} Export the booster and fins for 3D printing

  • Right click on the booster and fins
  • Select "Export..."
  • Select STL from the "Format" list
  • Depending on your 3D printer's program, you may want to set Units to Millimeter
  • Click OK
{% include tutorial-video.html path="videos/assembly/export_booster_and_fins" %} {% include section.html %}

{% include step_counter.html %} Export the fuselage for 3D printing

  • Right click on the fuselage
  • Select "Export..."
  • Select STL from the "Format" list
  • Depending on your 3D printer's program, you may want to set Units to Millimeter
  • Click OK
{% include tutorial-video.html path="videos/assembly/export_fuselage" %} {% include section.html %}

{% include step_counter.html %} Export the nosecone for 3D printing

  • Right click on the nosecone
  • Select "Export..."
  • Select STL from the "Format" list
  • Depending on your 3D printer's program, you may want to set Units to Millimeter
  • Click OK
{% include tutorial-video.html path="videos/assembly/export_nosecone" %}

{% include step_counter.html %} 3D print the files and gather parts

  • Printing will take about 3 hours
    • or use a 3rd party printing service (≈ $40)
  • Acquire:
    • Rocket engines. A, B, or C sizes will fit. A8-3 are a good model to start with (≈ $10)
    • Kit containing: (≈ $25)
      • launch pad
      • controller
    • Flame resistant flagging streamer (≈ $7)
    • 1/16 inch paracord / nylon shock coord (≈ $9)

{% include section.html %}

{% include step_counter.html %} Start assembling the rocket

  • Thread a flame-resistant string or shock cord through the tube inside the fuselage
{% include section.html %}

{% include step_counter.html %} Tie the string off on the bottom end with two knots to keep it from coming out

{% include section.html %}

{% include step_counter.html %} Tie the other end around the bar in the nosecone

  • You can use tweezers or a paperclip to get at the string when it's inside the nosecone.
{% include section.html %}

{% include step_counter.html %} Push the fuselage into the lower body. It should be a very tight fit to keep it from coming loose in flight.

  • Depending on your printer, you might have to sand a little bit off or put a piece of tape on the side to get the perfect fit.
{% include section.html %}

{% include step_counter.html %} Tie a streamer or parachute to the string

{% include section.html %}

{% include step_counter.html %} Roll the streamer and push it into the fuselage

{% include section.html %}

{% include step_counter.html %} Slip the nosecone on. It should be a loose fit so that the ejection charge will pop it off.

  • Depending on your printer, you might have to sand a little bit off or put a piece of tape on the side to get the perfect fit.
{% include section.html %}

{% include step_counter.html %} Load an engine (A, B, or C-class)

{% include section.html %}

{% include step_counter.html %} Go flying!
