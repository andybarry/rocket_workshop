{% include step_counter.html %} Rename your part studio

{% include adv-drawing.html path="img/tutorial/nosecone-drawing-1.png" %} {% include tutorial-video.html path="videos/nosecone/rename_nosecone" %} {% include written-instructions.html %}
  • Right-click on the Part Studio 1 tab on the bottom left
  • Select Rename

  • Rename it to "Nosecone"
    • You're already thinking of skipping this... but then your future self will be sad...
{% include section.html %}

{% include step_counter.html %} Create a sketch

{% include adv-drawing.html path="img/tutorial/nosecone-drawing-1.png" %} {% include tutorial-video.html path="videos/nosecone/create_sketch" %} {% include written-instructions.html %}
  • Select the sketch tool
  • Select a sketch plane. Use the Top plane.
  • Align your view to the sketch plane by clicking "Top" on the view cube
  • Make sure you have selected the Top plane. In the upper right, you should see "Top Plane" under sketch plane
{% include section.html %}

{% include step_counter.html %} Draw the outer body connector circle

{% include adv-drawing.html path="img/tutorial/nosecone-drawing-1.png" %} {% include tutorial-video.html path="videos/nosecone/nosecone_circle" %} {% include written-instructions.html %}
  • Select the circle tool
  • Draw a circle. Be sure to snap it to the origin.
{% include section.html %}

{% include step_counter.html %} Dimension the outer body connector circle

{% include adv-drawing.html path="img/tutorial/nosecone-drawing-1.png" %} {% include tutorial-video.html path="videos/nosecone/nosecone_circle_dimension" %} {% include written-instructions.html %}
  • Select the dimension tool
  • Click on your circle and set its diameter to 0.706
{% include section.html %}

{% include step_counter.html %} Draw the inner circle

{% include adv-drawing.html path="img/tutorial/nosecone-drawing-1.png" %} {% include tutorial-video.html path="videos/nosecone/inner_circle" %} {% include written-instructions.html %}
  • Select the circle tool
  • Draw a circle inside your first circle. Be sure to snap it to the origin.
  • Select the dimension tool
  • Click first on the outer circle and then on the inner circle and set the difference between them to 0.025
{% include section.html %}

{% include step_counter.html %} Extrude the connector

{% include adv-drawing.html path="img/tutorial/nosecone-drawing-1.png" %} {% include tutorial-video.html path="videos/nosecone/extrude_body_connector" %} {% include written-instructions.html %}
  • Select the extrude tool
  • ("Sketch 1" will automatically be selected)
  • Set the Depth to 0.6 in
  • Click the green check
