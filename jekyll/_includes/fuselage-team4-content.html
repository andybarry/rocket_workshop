{% include step_counter.html %} Start the sketch to add your name to the rocket

  • Start a new sketch
  • Select the front plane
  • Press n to line up your view (if upside down, click 'Front' on the view cube).
{% include tutorial-video.html path="videos/middle-body/text_sketch1" %}

{% include step_counter.html %} Add your name

  • Select the "Text" tool
  • Draw a box near your rocket and type your name with Enter after each letter
    • A
      N
      D
      Y
  • Select the Roboto Slab font
  • Press the green check
{% include tutorial-video.html path="videos/middle-body/text_sketch2" %}

{% include step_counter.html %} Constrain your name

  • Select the "Coincident" tool
  • Select the point that is the center of the top line above the first letter
  • Select the line running down the center of the rocket
    • You should see the letters jump to the center
{% include tutorial-video.html path="videos/middle-body/text_sketch3" %}

{% include step_counter.html %} Dimension your name

  • Press Esc to deselect the coincident tool
  • Drag the dashed line around the letters so that the top is below the top of the rocket
  • Select the "Dimension" tool
  • Select the top of the rocket
  • Select the top of the box around the first letter
  • Set the dimension to 0.25
{% include tutorial-video.html path="videos/middle-body/text_sketch4" %}

{% include step_counter.html %} Dimension your name (2)

  • Select the "Dimension" tool
  • Select the side of the box around the first letter
  • Set the dimension to whatever makes all the letters fit on the rocket. It's probably around 0.4
    • Make sure the letters don't go below the notch into the smaller diameter section
  • Press the green check to finish the sketch
{% include tutorial-video.html path="videos/middle-body/text_sketch5" %}

{% include step_counter.html %} Wrap your name

  • Select the "Wrap" tool (in a menu under the "Boolean" tool ).
  • Select Solid
  • Click on the box for Tools and select Sketch 4
  • Click on the box for Target and then click on the outside of the rocket body.
{% include tutorial-video.html path="videos/middle-body/text_wrap1" %}

{% include step_counter.html %} Wrap your name (2)

  • Right-click and drag the mouse to view the solid letters. If they are backwards, click the "Opposite direction" button next to Target.
  • Set Thickness to 0.02
  • Click the green check
{% include tutorial-video.html path="videos/middle-body/text_wrap2" %}

{% include step_counter.html %} Rename your part studio... you're done with the fuselage!

{% include tutorial-video.html path="videos/middle-body/rename_part_studio" %}
  • Right-click on the Part Studio 1 tab on the bottom left
  • Select Rename

  • Rename it to "Middle Body"
