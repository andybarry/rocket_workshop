4. Create a sketch

{% include tutorial-video.html path="videos/middle-body/create_sketch_top_plane" %}
  • Select the sketch tool
  • Select a sketch plane. Use the Top plane.
  • Align your view to the sketch plane by clicking "Top" on the view cube
  • Make sure you have selected the Top plane. In the upper right, you should see "Top Plane" under sketch plane
{% include section.html %}

5. Draw the outer body circle

{% include tutorial-video.html path="videos/middle-body/draw_outer_circle2" %}
  • Select the circle tool
    • Can't find it? »
    • You might have exited your sketch. Double click on "Sketch 1" to reenter the sketch.
  • Draw a circle. Be sure to snap it to the origin.
{% include section.html %}

6. Dimension the outer body circle

{% include tutorial-video.html path="videos/middle-body/draw_outer_circle_dimension" %}
  • Select the dimension tool
  • Click on your circle and set its diameter to 0.781 in
{% include section.html %}

7. Draw the inner circle

{% include tutorial-video.html path="videos/middle-body/draw_inner_circle" %}
  • Select the circle tool
  • Draw a circle inside your first circle. Be sure to snap it to the origin.
  • Select the dimension tool
  • Click first on the outer circle and then on the inner circle and set the difference between them to 0.025 in
{% include section.html %}

8. Extrude the body

{% include tutorial-video.html path="videos/middle-body/extrude_body" %}
  • Exit the sketch by clicking the green check
  • Select the extrude tool
  • Select "Sketch 1" from the left pane
  • Set the Depth to 3.0 in
  • Click the green check
