in which we extrude the body cylinder

{% include step_counter.html %} Create a new part studio

{% include tutorial-video.html path="videos/lower-body/create_part_studio" %}
  • Click the "+" button
  • Select Create Part Studio

{% include step_counter.html %} Rename your part studio

{% include tutorial-video.html path="videos/lower-body/rename_lower_body" %}
  • Right-click on the Part Studio 1 tab on the bottom left
  • Select Rename

  • Rename it to "Lower Body"
{% include section.html %}

{% include step_counter.html %} Create a sketch

{% include tutorial-video.html path="videos/lower-body/create_sketch" %}
  • Select the sketch tool
  • Select a sketch plane. Use the Top plane.
  • Align your view to the sketch plane by clicking "Top" on the view cube
  • Make sure you have selected the Top plane. In the upper right, you should see "Top Plane" under sketch plane

{% include step_counter.html %} Draw the outer body circle

{% include tutorial-video.html path="videos/lower-body/create_outer_body_circle" %}
  • Select the circle tool
  • Draw a circle. Be sure to snap it to the origin.

{% include step_counter.html %} Dimension the outer body circle

{% include tutorial-video.html path="videos/lower-body/dimension_outer_body_circle" %}
  • Select the dimension tool
  • Click on your circle and set its diameter to 0.781

{% include step_counter.html %} Draw the inner circle

{% include tutorial-video.html path="videos/lower-body/create_inner_body_circle" %}
  • Select the circle tool
  • Draw a circle inside your first circle. Be sure to snap it to the origin.
  • Select the dimension tool
  • Click first on the outer circle and then on the inner circle and set the difference between them to 0.025

{% include step_counter.html %} Extrude the body

{% include tutorial-video.html path="videos/lower-body/extrude_lower_body" %}
  • Select the extrude tool
  • Set the Depth to 4.0 in
  • Click the green check

{% include step_counter.html %} Use the camera to look at the body

{% include tutorial-video.html path="videos/lower-body/look_at_extrude1" %}
  • Hold the right mouse button to view your tube from different angles
  • Zoom in and out using the mouse wheel
