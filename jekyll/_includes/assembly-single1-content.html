{% include step_counter.html %} Insert parts

{% include adv-drawing.html path="img/tutorial/assembly-drawing-1.png" %} {% include tutorial-video.html path="videos/assembly/insert_parts" %} {% include written-instructions.html %}
  • Select the "Insert" tool
  • Select the "Middle body"
  • Click inside the assembly to place the part
  • Repeat for the "Lower body" and "Nosecone"
  • Click the green check
{% include section.html %}

{% include step_counter.html %} Mate the fuselage to the booster and fins

{% include adv-drawing.html path="img/tutorial/assembly-drawing-1.png" %} {% include tutorial-video.html path="videos/assembly/mate_booster_fuselage" %} {% include written-instructions.html %}
  • Select the "Fastened Mate" tool
  • Hover over the top of the booster and fins, then click when a disc appears
  • Hover over the inside of the fuselage, where the part gets wider
  • Click again, and you'll see the parts snap together
  • Click the green check
{% include section.html %}
