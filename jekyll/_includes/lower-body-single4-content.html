{% include step_counter.html %} Create a plane representing the top of the engine

We need to make a stopper so the engine doesn't explode out of the top of the rocket.
{% include adv-drawing.html path="img/tutorial/fins-booster-drawing-6.png" %} {% include tutorial-video.html path="videos/lower-body/engine_stop_plane" %} {% include written-instructions.html %}
  • Start a new plane with the plane tool
  • Select the bottom face of the rocket
  • If needed, click the "Opposite direction" button to make the plane appear in the middle of the rocket
  • Set the "Offset distance" to 2.9
  • Click the green check
{% include section.html %}

{% include step_counter.html %} Start a sketch on that plane

{% include adv-drawing.html path="img/tutorial/fins-booster-drawing-6.png" %} {% include tutorial-video.html path="videos/lower-body/engine_stop_sketch" %} {% include written-instructions.html %}
  • Select the sketch tool
  • Click "Plane 1"
  • Select the "Top" of the view cube to align your view with the sketch
{% include section.html %}

{% include step_counter.html %} Sketch the engine block

{% include adv-drawing.html path="img/tutorial/fins-booster-drawing-6.png" %} {% include tutorial-video.html path="videos/lower-body/engine_stop_circle" %} {% include written-instructions.html %}
  • Select the circle tool
  • Start the circle on the center point
  • Draw a circle smaller than the rocket
  • Use the dimension tool to set the diameter to 0.6
{% include section.html %}

{% include step_counter.html %} Add the rocket's body to your sketch

{% include adv-drawing.html path="img/tutorial/fins-booster-drawing-6.png" %} {% include tutorial-video.html path="videos/lower-body/engine_stop_use" %} {% include written-instructions.html %}
  • Turn your view slightly
  • Select the Use (Project/Convert) tool
  • Click the inner circle of the rocket
  • You should see that line bold as it becomes part of your sketch
{% include section.html %}

{% include step_counter.html %} Extrude the engine stop

{% include adv-drawing.html path="img/tutorial/fins-booster-drawing-6.png" %} {% include tutorial-video.html path="videos/lower-body/engine_stop_extrude" %} {% include written-instructions.html %}
  • Select the extrude tool
    • Your sketch will automatically be selected
  • If needed, click the "Opposite direction" button to make the extrude go upwards
  • Set the "Depth" to 0.5
  • Click the green check

{% include step_counter.html %} Start engine clip sketch

We need a clip to stop the engine from exploding out of the back of the rocket when the recovery (reverse) charge goes off. {% include adv-drawing.html path="img/tutorial/fins-booster-drawing-7.png" %} {% include tutorial-video.html path="videos/lower-body/engine_clip_sketch" %} {% include written-instructions.html %}
  • Select the sketch tool
  • Start a sketch on the bottom face of the rocket
  • Select the Use (Project/Convert) tool
  • Click the inner circle of the rocket
    • You should see that line bold as it becomes part of your sketch
  • Align your view with the sketch by clicking the Bottom of the view cube
{% include section.html %}

{% include step_counter.html %} Sketch the engine clip arc

{% include adv-drawing.html path="img/tutorial/fins-booster-drawing-7.png" %} {% include tutorial-video.html path="videos/lower-body/engine_clip_3_point_arc" %} {% include written-instructions.html %}
  • Select the 3-point arc tool
  • Set the points:
    1. Edge of the circle, to the left
    2. Edge of the circle, to the right
    3. Set the radius so the arc curves towards the rocket's center
{% include section.html %}

{% include step_counter.html %} Dimension the engine clip arc

{% include adv-drawing.html path="img/tutorial/fins-booster-drawing-7.png" %} {% include tutorial-video.html path="videos/lower-body/dimension_engine_clip" %} {% include written-instructions.html %}
  • Select the dimension tool
  • Click the outer circle and then the arc and set the distance to 0.04
  • Use the dimension tool again to set the arc's radius to 0.1
{% include section.html %}

{% include step_counter.html %} Constrain the engine clip arc

{% include adv-drawing.html path="img/tutorial/fins-booster-drawing-7.png" %} {% include tutorial-video.html path="videos/lower-body/constrain_engine_clip" %} {% include written-instructions.html %}
  • Select the "Vertical" tool
  • Click the arc's center-point and then the origin of the rocket
  • You should see the arc jump to the center
{% include section.html %}
