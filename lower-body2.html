Make the fins

in which we circular pattern the rocket fins

8. Start a sketch on the Right plane

  • Select the sketch tool
  • Select a sketch plane. Use the Right plane.
  • Align your view to the sketch plane by clicking "Right" on the view cube

9. Draw the profile of one fin

  • Select the Use (Line) tool
  • Start at the bottom, in the middle of the cylinder, and draw the profile of one fin.
  • The exact fin shape can be up to you!
  • Your fin should have these properties though:
    • A flat bottom
    • About as wide/tall as these fins


10. Extrude the fin in both directions

  • Select the extrude tool
  • Click on Add to add to your part instead of creating a new part
  • Set the Depth to 0.05 in
  • Check Second end position
  • Set the second Depth to 0.05 in
  • Click the green check

11. Add a fillet to smooth out the fin profile

  • Select the fillet tool
  • Set the radius to 0.4
  • Click all of edges of your fins except the bottom!
  • It's important to have the bottom be flat so it will print well
  • Click the green check

12. Add lots of more fillets to the fin

  • Select the fillet tool
  • Set the radius to 0.05
  • Click all of edges of your fins except the bottom!
  • It's important to have the bottom be flat so it will print well
  • Click the green check

11. Turn on section view

  • We want to see inside the tube, so we'll have our view cut away part of the tube. This does not change the part, just how we look at it
  • Select the view menu
  • Select "Turn section view on"
  • Click on the Right plane
  • Click the green check

12. Fillet the body connector ring

  • Select the fillet tool
  • Click on the outside ring of the ledge
  • Click the green check
