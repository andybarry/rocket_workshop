Getting started

Camera controls and a cube

We recommend having this tutorial open on one side of your screen, and Onshape open on the other side:

1. Create an onshape account

  • Head to the create account page at Onshape
  • Create an account
  • Click the activation link in your email

2. Create a new document

  • Sign-in to onshape
  • Click the Create button
  • Name your document Box by [yourname]
  • Select Create public document


3. Moving your view in 3D

  • Turn your view: right-click + drag
  • Pan/slide: Middle-button + drag to pan/slide your view
  • Zoom: scroll wheel


4. Create a sketch

  • Select the sketch tool
  • Select a sketch plane. Use the Top plane.
  • Align your view to the sketch plane by clicking "Top" on the view cube

5. Draw the outer body connector circle

  • Select the circle tool
  • Draw a circle. Be sure to snap it to the origin.
  • Select the dimension tool
  • Click on your circle and set its diameter to 0.707

6. Draw the inner circle

  • Select the circle tool
  • Draw a circle inside your first circle. Be sure to snap it to the origin.
  • Select the dimension tool
  • Click first on the outer circle and then on the inner circle and set the difference between them to 0.025

7. Extrude the connector

  • Exit the sketch by clicking the green check
  • Select the extrude tool
  • Select "Sketch 1" from the left pane
  • Set the Depth to 0.6 in
  • Click the green check
