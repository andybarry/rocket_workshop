# Drone Workshop Instructions

Interactive slide viewer for the Drone Workshop.

## Features

- 6 interactive instruction slides
- Left/Right navigation arrows
- Keyboard navigation support (arrow keys)
- Slide indicator dots
- Responsive design
- Beautiful gradient UI

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Visit: `http://localhost:5174/drone/`

## Structure

- `src/DroneInstructions.jsx` - Main slide viewer component
- `src/DroneInstructions.css` - Styling
- `public/workshop-slides/` - PNG slides (1-6)

## Usage

Navigate through slides using:
- Arrow buttons at the bottom
- Dot indicators
- Keyboard arrow keys (← →)
