Time to make rockets!

We recommend having this tutorial open on one side of your screen, and Onshape open on the other side

1. Open the shared file

2. Create a new part studio

  • Click the "+" button
  • Select Create Part Studio

3. Rename your part studio

  • Name it "Middle Body"
    • You're already thinking of skipping this... don't.

4. Create a sketch

  • Select the sketch tool
  • Select a sketch plane. Use the Top plane.

5. Draw the body circles

  • Select the circle tool
  • Draw two concentric circles. Be sure to snap them to the origin.

6. Dimension the circles

  • Select the dimension tool
  • Click on the outer circle and set its diameter to 0.781
  • Click first on the outer circle and then on the inner circle and set the difference between them to 0.025
