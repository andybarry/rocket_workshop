Check if the Driver is Installed

  1. Open Device Manager: Press Windows + X and select “Device Manager.”
  2. Expand Ports (COM & LPT): Click the ▶︎ next to “Ports (COM & LPT).”
  3. Look for Silicon Labs: You should see “Silicon Labs CP210x USB to UART Bridge.” If it’s there, the driver is installed.
  4. If you don’t see it:
    • Download and install the CP210x driver from Silicon Labs: Silicon Labs CP210x Driver
