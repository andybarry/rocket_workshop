13. Create a plane for the nosecone tether sketch

  • Start a new plane with the plane tool
  • Select the top face of your tube.
  • Set the offset to 1.25 in
  • Click the "Opposite direction" button to make your plane fall inside your tube
  • Click the green check

14. Start the sketch for the nosecone tether holder

  • Start a new sketch
  • Select the plane you just created
  • Align your view to the sketch plane by clicking "Top" on the view cube
  • Select the "Use (Project/Convert)" tool

15. Sketch the outside of the tether holder

  • Select the "3 point arc" tool
  • Set the first point on the circle, above the centerline
  • Set the second point on the circle, below the centerline
  • Set the third point so the arc is in inside the tube

16. Constrain the arc's center to the circle

  • Select the "Coincident" tool
  • Select the point that is the center of the arc you just created
  • Select the circle, and you'll see the point jump onto the circle

17. Constrain the arc's center to the centerline

  • Select the "Horizontal" tool
  • Select the arc's center
  • Select the origin point, and you'll see the point jump to the line and turn black (black = fully constrained)


18. Draw the center circle

  • Select the circle tool
  • Draw a circle inside your arc and the tube (make it small)
  • Constrain the circle's center to be horizontal with the origin, as you did above

19. Dimension the sketch

  • Dimension the circle's gap:
    • Select the dimension tool
    • Click the circle's edge
    • Click the tube's circle that you have been using
    • Set the dimension to 0.01
  • Dimension the gap on the other side (with the dimension tool still selected):
    • Click the circle's edge
    • Click your 3-point arc
    • Set the dimension to 0.025
  • Dimension the circle's diameter (with the dimension tool still selected):
    • Select the circle
    • Set the dimension to 0.13


20. Finish the sketch and extrude the holder

  • Select the "Trim" tool
  • Select the part of the circle outside of the arc
  • Click the green check
  • Select the extrude tool
  • Select your newly created sketch
  • Set the depth to 0.75
  • Click the "Opposite direction" to make the extrude go towards the bottom of the part
  • Click the green check

21. Fillet the tether holder

  • Select the "fillet" tool
  • Click the arc on the edge of the tube and the holder
  • Set the radius to 0.15
  • Select the same arc on the bottom of the holder
  • Click the green check

22. Exit section view... you're done!

  • Select the view cube menu
  • Select "Turn section view off"
